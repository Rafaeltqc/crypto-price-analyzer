# Crypto Price Analyzer

Projeto em Python que permite analisar o preço de criptomoedas através do download de dados históricos e da geração de gráficos de preço.

O programa utiliza dados financeiros obtidos online e cria automaticamente um gráfico mostrando a evolução do preço da criptomoeda escolhida.

---

# 📊 Funcionalidades

- Escolher qual criptomoeda deseja analisar
- Baixar automaticamente dados históricos de preço
- Gerar um gráfico com a evolução do preço ao longo do tempo

---

# 🛠 Tecnologias utilizadas

Este projeto foi desenvolvido utilizando:

- Python
- pandas
- matplotlib
- yfinance

Essas bibliotecas são utilizadas para:

- obter dados financeiros
- manipular os dados
- gerar gráficos de visualização

## 🚀 Como executar o projeto

### 1. Clonar o repositório

Abra o terminal e execute:

```
git clone https://github.com/Rafaeltqc/crypto-price-analyzer.git
```

### 2. Entrar na pasta do projeto

```
cd crypto-price-analyzer
```

### 3. Instalar as dependências

```
pip install -r requirements.txt
```

### 4. Executar o programa

```
python crypto_analyzer.py
```
